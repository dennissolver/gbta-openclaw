# EasyOpenClaw

> User-friendly wrapper for OpenClaw autonomous AI agent

EasyOpenClaw simplifies AI automation by providing a web-based interface, onboarding flow, and project management layer around the open-source OpenClaw agent. It eliminates technical complexity while adding enterprise features like user isolation, voice interaction, and template-based workflows.

**Built for**: Businesses and professionals who want AI automation without technical setup - including marketers, content creators, researchers, and teams needing automated workflows
**Industry**: DevTools (B2B, SaaS, API-first, Enterprise)
## Tech Stack

EasyOpenClaw is built with:

- **Framework**: Next.js
- **Language**: JavaScript
- **Database**: PostgreSQL
- **Authentication**: Supabase Auth
- **Deployment**: Vercel
- **Services**: Supabase, ElevenLabs
